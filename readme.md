-Bot de Citas Consulares

Este proyecto automatiza la búsqueda de citas en una página consular y envía notificaciones a Telegram cuando hay disponibilidad.

---Funcionalidades
Login automático
Navegación automática en la página
Detección de citas disponibles
Notificaciones por Telegram

---Tecnologías
JavaScript
Tampermonkey
Telegram Bot API

---Instalación
Instala Tampermonkey en tu navegador
Crea un nuevo script
Copia el contenido de script.user.js
Configura tus datos:
const TOKEN = "TU_TOKEN";
const CHAT_ID = "TU_CHAT_ID";


---Nota

No compartas tus credenciales ni tokens.

---Uso

Abre la página de citas y el script hará el resto automáticamente.
